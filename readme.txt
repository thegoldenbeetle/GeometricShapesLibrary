Немодифицирующий алгоритм:
Поиск двух смежных элементов, равных между собой по определенному критерию.
В качестве критерия поиска использую цвет фигуры.

Модифицирующий алгоритм:
Создать упорядоченное объединение двух диапазонов.
В качестве критерия сортировки использую значение периметра фигуры.

Реализация алгоритмов находится в файле oop_lr5.h
В файле Unittest_shapes приведены тесты, подтверждающие корректность работы написанных функций.
В файле oop_lr5.cpp приведен код, демонстрирующий работу написанных функций.
